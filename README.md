# CodeBridge AI — Site Oficial v1.0

> "Da IA ao software de verdade."

## Como usar

1. Extraia o ZIP
2. Abra `index.html` no navegador (ou use **Live Server** no VS Code)
3. Nenhum build necessário — HTML, CSS e JS puros

## Estrutura

```
codebridge-ai/
├── index.html           ← Homepage principal
├── about.html           ← Sobre a empresa
├── founders.html        ← Hélio e Eric
├── services.html        ← Serviços e preços
├── brand-center.html    ← Identidade visual
├── idea-hub.html        ← Roadmap e ideias
├── resources.html       ← Guias e templates
├── blog.html            ← Blog técnico
├── community.html       ← Comunidade
├── contact.html         ← Formulário de contato
└── assets/
    ├── css/
    │   ├── variables.css   ← Tokens de design
    │   ├── style.css       ← Estilos globais
    │   ├── components.css  ← Componentes (nav, cards, footer...)
    │   └── responsive.css  ← Mobile-first
    └── js/
        ├── main.js         ← Scroll, reveal, counters
        ├── navigation.js   ← Header e menu mobile
        └── animations.js   ← Barras e animações
```
